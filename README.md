# Smart Weather-Adaptive Drying Solution (IoT)

An autonomous mobile system designed to solve outdoor laundry difficulties through real-time environmental monitoring and automated response logic.

## 🛠 Tech Stack
* **Hardware:** Raspberry Pi 5, L298N Motor Driver, BME280 (Humidity/Temp), BH1750 (Light), MPU6050 (Motion), HC-SR04 (Ultrasonic).
* **Software:** Python (System Logic), Dart/Flutter (Mobile App).
* **Cloud/Data:** Firebase Realtime Database.
* **Protocols:** I2C, GPIO Interrupt Handling.

## 🌟 Key Features
* **Autonomous Navigation:** Real-time obstacle avoidance and movement control using PID-style logic.
* **Environmental Sensing:** Continuous monitoring of rain, light intensity, and humidity to automate drying positions.
* **Remote Command:** Full manual override and data visualization via a custom-built Flutter mobile application.
* **Safety Logic:** Integrated GPIO interrupt handling for motor encoders ensuring precise and safe mechanical movement.
